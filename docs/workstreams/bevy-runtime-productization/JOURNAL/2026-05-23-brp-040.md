# BRP-040 Editor Shell Productization

Date: 2026-05-23
Status: DONE

## Summary

Converted `backends/dear-imgui-bevy/examples/editor_shell.rs` from a one-frame proof to a
persistent Bevy `DefaultPlugins` windowed editor shell.

The example now:

- opens a real window and exits on Escape;
- renders a live offscreen 2D scene to a `Handle<Image>`;
- registers the image through `ImguiBevyTextures` and displays it as an ImGui viewport;
- uses docked hierarchy, inspector, input policy, and diagnostics panels;
- tracks viewport zoom, playback state, selection state, capture flags, and frame diagnostics.

Added `render::ImguiOverlayDisabled` as a small backend helper marker so the offscreen scene camera
can opt out of the global ImGui overlay pass.

## Coverage

- `tests/render_extract.rs` now covers disabled overlay cameras being excluded from extracted
  overlay targets.
