# BRP-020 — Persistent Windowed Runtime Proof

Added the first persistent runtime Bevy example for the backend.

## Completed Slice

- Added `backends/dear-imgui-bevy/examples/windowed_overlay.rs`.
- Added a dev-only top-level `bevy = "=0.19.0-rc.2"` dependency with `2d` and
  `default_platform` features for the example.
- Registered the example in `backends/dear-imgui-bevy/Cargo.toml` with `required-features =
  ["render"]`.
- Updated the backend README with the manual run command.

## Validation

- `cargo +stable check -p dear-imgui-bevy --features render --example windowed_overlay` — PASS.

## Notes

The example uses Bevy `DefaultPlugins` and the normal windowed runner. It opens a real OS window,
draws ImGui overlay state across frames, and exits when Escape is pressed. It is a manual runtime
smoke app rather than an automated CI execution gate.
