# 2026-05-22 — BEVY-020 Core Lifecycle Proof

## Summary

Implemented the first engine-managed Dear ImGui frame lifecycle slice for Bevy-native integration. The core API now lets an engine prepare IO, open exactly one frame, run multiple sequential UI systems through a shared `Ui`, close/render once, and optionally build a thread-safe `FrameSnapshot` at the frame boundary.
