## Summary

Hardened the Bevy backend runtime smoke coverage against real native render resources.

The current proof uses the existing ignored GPU harness in `backends/dear-imgui-bevy/src/render.rs`
to drive a real `RenderDevice` and `RenderAssets<GpuImage>` path. It proves the backend can prepare
texture bind groups from actual Bevy GPU images and can reject non-sampled images instead of
silently accepting them.
